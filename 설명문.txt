C# static ThreadLocal
: 스레드 로컬 스토리지의 이해
C# static ThreadLocal이란?
C#에서 static ThreadLocal은 각 스레드마다 독립적인 값을 저장하는 데 사용되는 메커니즘입니다. 
쉽게 말해, 여러 스레드가 동시에 실행될 때, 
각 스레드는 자신만의 고유한 값을 가지고 작업할 수 있도록 해주는 것이죠.

static ThreadLocal의 역할
스레드 안전성 보장: 여러 스레드가 공유하는 변수에 대한 동기화 문제를 해결하여 스레드 
안전성을 보장합니다.
스레드별 데이터 관리: 각 스레드는 자신만의 값을 가지므로 다른 스레드의 영향을 받지 않고 
독립적으로 작업할 수 있습니다.
전역 변수의 스레드별 분리: 전역 변수를 스레드별로 분리하여 
사용할 수 있도록 해줍니다.
